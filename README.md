# Clean Architecture Sample Project

![.NET Core](https://github.com/nebioo/Clean/workflows/.NET%20Core/badge.svg)

## Api
  -asp.net core 5.0
  
  -MediatR, CQRS, Generic Repository
  
  -jwt auth
  
  -ef core
  
  -azure blob storage
  
## Database
  -postgresql 

## Api usage
open project folder
```bash
cd Api\
```
```bash
dotnet restore 
dotnet run
```

open https://localhost:5001/swagger/index.html in browser
